隐私政策

1. 信息收集

我们可能会收集以下信息：

  • 您的姓名、联系方式、电子邮件地址等个人信息。
  • 您的停车记录、车辆信息等使用数据。
  • 您的设备信息，如 IP 地址、浏览器类型等。

2. 信息使用

我们收集的信息将用于以下目的：

  • 提供和改进我们的服务。
  • 与您联系，发送重要通知。
  • 进行数据分析，优化用户体验。

3. 信息共享

我们不会将您的个人信息出售或出租给第三方。但在以下情况下，我们可能会共享信息：

  • 根据法律法规的要求。
  • 为保护我们的合法权益。
  • 在您同意的情况下。

4. 信息安全

我们采取合理的技术和管理措施，保护您的个人信息安全。

5. 隐私政策的变更

我们可能会不时更新本隐私政策。更新后的政策将在网站上公布。

服务条款

1. 服务内容

我们为您提供智慧停车管理系统及相关服务，包括但不限于停车位预订、停车记录查询等功能。

2. 用户责任

您在使用我们的服务时，需遵守以下规定：

  • 不得利用我们的服务从事非法活动。
  • 不得干扰或破坏我们的系统和服务。
  • 确保您提供的信息真实、准确。

3. 服务变更与终止

我们保留随时修改、暂停或终止服务的权利，恕不另行通知。

4. 免责声明

我们不对以下情况承担责任：

  • 因不可抗力导致的服务中断。
  • 因用户不当使用服务导致的损失。
  • 第三方行为导致的损失。

5. 法律适用

本服务条款受中华人民共和国法律管辖。
